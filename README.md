# OpenCV Camera Playground

A real-time computer vision project built with Python and OpenCV.

## Overview

This project captures live video from a webcam and applies image processing algorithms in real time. It was created as part of my journey into computer vision and OpenCV.

The application supports multiple processing modes and displays the current FPS (frames per second), allowing users to see both the visual effect and the performance of the system.

## Features

* Live webcam feed
* Real-time grayscale conversion
* Real-time edge detection using the Canny algorithm
* FPS counter
* Keyboard controls for switching modes
* Camera resource management and safe shutdown

## Controls

| Key | Action              |
| --- | ------------------- |
| 1   | Normal camera mode  |
| 2   | Grayscale mode      |
| 3   | Edge detection mode |
| q   | Quit application    |

## Technologies Used

* Python
* OpenCV
* Computer Vision Fundamentals

## Installation

Clone the repository:

```bash
git clone https://github.com/anhelinakhpi/opencv-camera-playground.git
cd opencv-camera-playground
```

Install dependencies:

```bash
pip install opencv-python
```

Run the project:

```bash
python main.py
```
